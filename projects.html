Task Manager

I created a task management web app with list and kanban views for a project in college.

The focus of the course was enabling civil engineering students to be able to put information on the web, so there was very little CSS formatting and I have since learned that some of the ways we were taught to do things were not robust enough for production code. I've spent some time fixing it up to help it reflect my current skills. The master branch is the latest fully working version, but please refer to the 'fix/POST_PATCH_Requests' branch for better code (although not everything is updated yet).

You can view the project on Github here. Follow the instructions in the readme to download and run the project.
